scaleme.js documentation

A tiny, dependency-free charting library. Three engines: data → scales → pixels.

Install

npm install scaleme.js

Then import what you need. Everything is a named export.

import { createChart, createPartograph } from "scaleme.js";

Quick start

Give createChart a chart type, a mount target, and an array of { x, y } points. That's it. 

import { createChart } from "scaleme.js";

createChart({
  type: "line",        // or "bar"
  element: "#app",     // CSS selector, container, or <canvas>
  data: [
    { x: 1, y: 10 },
    { x: 2, y: 20 },
    { x: 3, y: 15 },
  ],
});

Pass a container element and scaleme creates a responsive <canvas> inside it. Pass a <canvas> directly and it draws into that. Rendering is high-DPI and repaints on window resize.

Line charts

createChart({
  type: "line",
  element: "#chart",
  color: "#0ea5e9",
  smooth: true,      // curved line
  area: true,        // fill underneath
  points: false,     // hide dots
  lineWidth: 3,
  data: myPoints,
});

Bar charts

Bars center on their x value and grow from the zero baseline. If your data goes negative, bars below zero grow downward automatically. 

createChart({
  type: "bar",
  element: "#chart",
  color: "#f59e0b",
  barRadius: 8,      // rounded corners
  barPadding: 0.8,   // fatter bars
  data: [
    { x: 1, y: 8 }, { x: 2, y: 22 }, { x: 3, y: 15 },
  ],
});

Styling recipes

Minimal sparkline

createChart({
  type: "line", element: "#spark",
  axes: false, grid: false, points: false, smooth: true,
  data: myPoints,
});

Dark themed panel

createChart({
  type: "line", element: "#chart",
  color: "#22d3ee",
  background: "#0f172a",
  area: true, areaColor: "rgba(34, 211, 238, 0.18)",
  smooth: true,
  data: myPoints,
});

The chart handle

createChart returns a handle for live updates:

const chart = createChart({ type: "line", element: "#app", data });

chart.update(newData); // swap data + repaint
chart.render();        // repaint (also auto-fires on resize)
chart.destroy();       // remove listeners

createPartograph

The WHO labour-monitoring chart this library was first built for. It stacks clinical panels — fetal heart rate, the cervicograph (with Alert & Action lines), contractions, pulse/BP, and more — on one shared time axis. See the live partograph demo. 

import { createPartograph } from "scaleme.js";

createPartograph({
  element: "#partograph",
  hours: 12,
  alertLineStartHour: 0,
  patient: { name: "A. Mother", gravida: "2", para: "1" },
  fetalHeartRate: [{ hour: 0, value: 140 }, { hour: 1, value: 136 }],
  cervix:  [{ hour: 0, value: 4 }, { hour: 4, value: 6 }],  // plotted ✕
  descent: [{ hour: 0, value: 4 }, { hour: 4, value: 3 }],  // plotted ◯
  contractions: [{ hour: 4, count: 4, intensity: "strong" }],
  amnioticFluid: [{ hour: 0, value: "C" }],   // I·C·M·B·A
  moulding: [{ hour: 0, value: "+" }],        // 0·+·++·+++
  pulse: [{ hour: 0, value: 80 }],
  bloodPressure: [{ hour: 0, systolic: 120, diastolic: 80 }],
  temperature: [{ hour: 0, value: 36.8 }],
});

Low-level: scale & renderer

The engines are exported too, if you want to compose your own chart (that's exactly how the partograph is built). 

import { linearScale, canvasRenderer } from "scaleme.js";

// value -> pixel, and back
const x = linearScale([0, 100], [0, 400]);
x.map(50);       // 200
x.invert(200);   // 50
x.ticks(5);      // [0, 20, 40, 60, 80, 100]

// draw with high-level primitives
const r = canvasRenderer(ctx, width, height);
r.polyline(points, { stroke: "#2563eb", width: 2, smooth: true });
r.roundRect(x, y, w, h, 6, "#16a34a");
r.area(points, baselineY, "rgba(37,99,235,0.15)");
